## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/manumang29/SpringAIDemo.git
   ```
2. Navigate to the project directory:
   ```bash
   cd SpringAIDemo
   ```
3. Build the project:
   ```bash
   ./mvnw clean install
   ```
4. Run the application:
   ```bash
   ./mvnw spring-boot:run
   ```
